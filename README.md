# PokeDex Explorer

## Descripción

PokeDex Explorer es una aplicación web desarrollada con React y Tailwind CSS que permite explorar información detallada sobre los Pokémon. La aplicación muestra estadísticas, tipos y características de cada Pokémon de forma clara y profesional, con un diseño responsivo y elegante.

El proyecto incluye un listado de Pokémon destacados en la página principal y un grid completo con todos los Pokémon disponibles a través de la API pública PokeAPI.

---

## Tecnologías usadas

* **React**: Biblioteca para construir interfaces de usuario.
* **React Router DOM**: Manejo de rutas y navegación.
* **Tailwind CSS**: Framework de CSS para diseño responsivo y limpio.
* **Vite**: Herramienta de construcción rápida y moderna.
* **Lucide-React**: Iconos modernos y ligeros.
* **PokeAPI**: API pública para obtener datos de Pokémon.

---

## Pasos para ejecutar el proyecto

1. **Clonar el repositorio:**

```bash
git clone https://github.com/josep-rivera/dae-evaluacion-3.git
```

2. **Ingresar a la carpeta del proyecto:**

```bash
cd dae-evaluacion-3
```

3. **Instalar dependencias:**

```bash
npm install
```

4. **Iniciar el servidor de desarrollo:**

```bash
npm run dev
```

5. **Abrir la aplicación en el navegador:**

Por defecto, Vite abre la aplicación en `http://localhost:5173`.
